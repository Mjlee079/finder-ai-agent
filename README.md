# Finder AI Agent

Finder AI Agent is a project scaffolded from the local `finderAi` workspace.

## Contents

- `requiremenrs.txt` – Python dependencies for the backend (Flask, Torch, Ultralytics, OpenCV, etc.).

## Setup

1. Create and activate a Python virtual environment.
2. Install dependencies:

```bash
pip install -r requiremenrs.txt
```

3. Run your backend and/or frontend following your project’s structure.
